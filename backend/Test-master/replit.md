# Test

## Overview
A minimal static web project imported from GitHub. The repository contained only a README with the word "Test" — no application code, framework, or dependencies were present.

## Project Structure
- `index.html` — Simple HTML page served as the frontend
- `server.py` — Python HTTP server serving the static files on port 5000
- `README.md` — Original repo README

## Running the App
The workflow "Start application" runs `python server.py`, which serves `index.html` on `0.0.0.0:5000`.

## Deployment
Configured as a static deployment serving the root directory.
