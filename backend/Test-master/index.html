```
📚 API Documentation

    

      

        
GET /api/parts

        
Description: Search for parts by device name

        
Query Parameters:

        ?device=iPhone17PrM
        
Response:

        
{
```

“success”: true,
“device”: “iPhone 17 Pro Max”,
“count”: 3,
“parts”: [
{
“skuId”: “IF17PrM-3-MOD-BAT”,
“partName”: “Replacement Battery”,
“category”: “Battery”,
“price”: 25.00,
“stock”: 150,
“quality”: “Premium”
}
]
}




```
      

        
GET /api/inventory/:skuId

        
Description: Get detailed information about a specific part

        
Example:

        GET /api/inventory/IF17PrM-3-MOD-BAT
        
Response:

        
{
```

“success”: true,
“part”: {
“skuId”: “IF17PrM-3-MOD-BAT”,
“partName”: “Replacement Battery”,
“category”: “Battery”,
“price”: 25.00,
“stock”: 150,
“quality”: “Premium”,
“specifications”: “Lithium-Ion, Adhesive…”,
“compatibleModels”: []
}
}




```
      

        
GET /api/compatibility/:skuId

        
Description: Get all devices compatible with a specific SKU

        
Example:

        GET /api/compatibility/IF13-14-1-DIS-OLED
        
Response:

        
{
```

“success”: true,
“skuId”: “IF13-14-1-DIS-OLED”,
“compatibleModels”: [
{
“id”: 5,
“modelNumber”: “A2482”,
“marketingName”: “iPhone 13”
},
{
“id”: 6,
“modelNumber”: “A2649”,
“marketingName”: “iPhone 14”
}
]
}




```
      

        
GET /api/brands

        
Description: Get all brands in the system

        
Response:

        
{
```

“success”: true,
“count”: 2,
“brands”: [
{ “id”: 1, “name”: “Apple” },
{ “id”: 2, “name”: “Samsung” }
]
}




```
      

        
GET /api/models

        
Description: Get all models, optionally filtered by brand

        
Query Parameters:

        ?brandId=1
        
Response:

        
{
```

“success”: true,
“count”: 6,
“models”: [
{
“id”: 1,
“brandId”: 1,
“modelNumber”: “A3257”,
“marketingName”: “iPhone 17 Pro Max”,
“releaseYear”: 2025
}
]
}
